運算思維 (Computational Thinking)

1. 拆解 (Decomposition)

把大怪獸（複雜問題）切成一小塊一小塊的任務。例如：要畫一個圓，先想怎麼畫一小段弧線。

2. 規律 (Pattern Recognition)

找出這幾塊任務裡，有沒有「長得很像」的地方？有的話，我們之後可以用迴圈處理。

3. 抽象 (Abstraction)

抓重點！把不影響結果的細節丟掉。例如：算總分時，名字不重要，分數才是重點。

4. 演算法 (Algorithm)

寫出 SOP。第一步讀取資料、第二步計算、第三步印出。只要步驟對了，電腦就不會出錯！

變數命名規則

Python 命名鐵律

  • 只能包含字母、數字、底線 ( _ )
  • 不能以數字開頭 (如 1st_place)
  • 不能使用保留字 (如 if, class)
  • 大小寫視為不同 (age vs Age)
  • 慣用 snake_case (蛇形命名法)
    「用底線連接單字，讓名稱更好讀，是 Python 社群的習慣寫法。」

觀念：變數 vs 函式

變數 (Variable)

像是一個「書包」或「標籤盒」，用來把資料存在裡面，方便以後拿出來用。

生活比喻 冰箱裡的雞蛋
編輯器顏色 通常是橘色或淡紅色
eggs = 3

函式 (Function)

像是一份「食譜」或「遙控器按鍵」，你呼叫它的名字，它就幫你執行動作。

生活比喻 炒飯的食譜
編輯器顏色 通常是淡藍色 (Aqua)
print("Hello")

編輯器的「顏色小密碼」

編輯器替不同角色的程式碼上色（語法高亮），是為了讓你一眼分辨誰是「存資料的（變數）」，誰是「做事情的（函式）」。 記住：括號 () 通常跟著函式出現，那是叫它「動起來」的訊號！

輸入 (Input)

# 基本用法
name = input("姓名？")
# 轉型為整數
age = int(input())
# 轉型為小數
h = float(input())

重要觀念小筆記

  • 1. 引號是「內容」的標記： "小明" 是內容，像一張寫了名字的紙； name 是盒子，用來裝那張紙。 ※ 沒加引號，電腦會去找一個叫這名字的變數盒子，但我們還沒宣告它，所以會出錯！
  • 2. input() 拿到的都是文字： 無論你輸入 100 還是 Hello，Python 一律先用「字串引號」把它們包起來。→ 例如你輸入 5，Python 拿到的其實是 "5"，沒辦法直接加法，要先用 int() 轉型喔！
  • 3. ★ 避開 int() 地雷： 如果你在測資看到 10 20 (中間有空格)，絕對不能直接用 int(input())，因為 Python 無法把「含空格的整行字」一次變成一個數字。 正確解法：先用 .split() 把空格切開，再分別轉成數字喔！

換行符號 \n 的妙用

想讓使用者在「下一行」輸入嗎？在提示字串最後加上 \n 吧！

ans = input("請輸入您的答案：\n")
# 執行時，游標會跳到下一行等待

解題實戰：如何處理「一行多個數字」？(解題網站 OJ 常用技巧)

# 萬用解題起手式 (必背!)
nums = list(map(int, input().split()))

這行程式碼是解題網站 (OJ) 最常用的寫法，能一次把「一行空白隔開的數字」全部轉成「整數串列」。
