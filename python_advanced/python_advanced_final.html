1.3 格式化輸出 (f-string)

🎯 學完這節，你會用 f-string 控制數字小數位數、補零、對齊，輸出更專業的格式。

在字串引號前加上 f 或 F，即可在字串中使用 {} 嵌入變數或運算式。冒號 : 後面接的是格式規則，讓你控制數字或文字的顯示方式。

📐 語法結構
f"{ 值 : 填充字元 對齊方式 寬度 . 精度 型別 }"
● 值：變數或數字 ● 填充字元：如 0 或空格 ● 對齊：^ < > ● 寬度：總字元數 ● 精度：小數位數 ● 型別：f / d / s
（以上選項皆為選填，依需求組合使用）
# 定義變數
name = "Alice"
score = 95.123
# 使用 f-string 格式化
msg = f"Hi, {name}. Score: {score:.2f}"
print(msg)

2.4 短路求值 (Short-circuit Evaluation)

Python 在計算 and / or 時，只要已經能確定結果，就不會繼續往後算。這就像電路短路一樣，電流不再繼續流下去。
💬 白話說：Python 會偷懶，確定答案後就不繼續算了——and 遇到 False 直接停、or 遇到 True 直接停。

or 運算 (A or B)

6.2 淺拷貝 vs 深拷貝

淺拷貝像是複印一本書的目錄，內容頁還是共用的——改了原本的內容，複印本也會跟著變。深拷貝像是把整本書連內容都重新印一份，兩本完全獨立，互不影響。
copy() 是淺拷貝（只複製第一層），deepcopy() 是深拷貝（全部都重新複製）。

* 深拷貝需要 `import copy` 模組。

7.1 格式化進階: 對齊、填充、千分位

使用 f-string 進行字串格式化，可控制對齊、填充字符及數字千分位。

7.2 正則表達式(Regular Expression)基礎

使用 re 模組進行複雜的字串匹配與操作。
常見函數：re.search(), re.findall(), re.sub()

import re
text = "My phone number is 123-456-7890."
pattern = r"\d{3}-\d{3}-\d{4}"
match = re.search(pattern, text)
if match:
  print(match.group()) # 輸出: 123-456-7890

7.3 字串編碼: Unicode、UTF-8

字串在 Python 3 預設為 Unicode。使用 .encode() 和 .decode() 進行編碼轉換。

s = "你好"
encoded_s = s.encode("utf-8")
print(encoded_s) # 輸出: b'\xe4\xbd\xa0\xe5\xa5\xbd'
decoded_s = encoded_s.decode("utf-8")
print(decoded_s) # 輸出: 你好

7.4 join() vs 重複串接(效能)

str.join() 對於拼接大量字串比使用 + 運算子效率更高。

words = ["Hello", "world", "from", "Python"]
# 使用 join (推薦)
sentence_join = " ".join(words)
print(sentence_join) # 輸出: Hello world from Python

# 使用 + 運算子 (效率較差)
sentence_concat = ""
for word in words:
  sentence_concat += word + " "
print(sentence_concat.strip()) # 輸出: Hello world from Python
