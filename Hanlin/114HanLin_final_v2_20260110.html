哈囉(input/print)

請設計一個程式，讓使用者輸入名字後，電腦會將名字呈現在畫面上與使用者打招呼。

計算學期成績 (if/else)

請設計一個程式，讓使用者輸入各項成績後，再將各項成績轉換為學期成績，並判斷學期成績是否及格？（其中，作業成績占 40％，測驗成績占 40％，平時成績占20％，學期成績 60 分為及格分數。）

執行結果 (模擬 Console):

等待計算...

對應程式碼邏輯：

# 使用 int() 將輸入的字串轉換成整數
x = int(input('Please enter homework grades:\n'))
y = int(input('Please enter quiz score:\n'))
z = int(input('Please enter your usual grades:\n'))

# 計算加權分數，結果為浮點數 (float)
grade = x * 0.4 + y * 0.4 + z * 0.2

# print() 使用逗號 (,) 分隔時，Python 會自動處理型態並加入空格
# 所以這裡不需要用 str() 轉換 grade
print('Semester grades are', grade ,'points.')

# 判斷及格與否 (注意縮排及冒號)
if grade < 60:
    # 條件成立時執行 (縮排)
    print('Failed!')
else:
    # 條件不成立時執行 (縮排)
    print('Pass!')

💡 觀念提示：

  • if 與 else 結尾都要有冒號 :
  • Python 使用縮排 (Indentation) 來區分程式區塊，通常為 1 個 Tab 鍵或 4 個空白鍵。

程式邏輯像是在數線上切蛋糕，if 先切走一塊，剩下的才輪到 elif 切，最後剩下的給 else。

進階邏輯範圍圖解 (以 18 到 25 為例)

觀察 and (且) 與 or (或) 在數線上的不同表現：

1. 介於 18 到 25 之間 (AND)

if age >= 18 and age <= 25:

累加計算 (for迴圈)

請設計一個程式，讓使用者輸入一個正整數 n，然後計算並顯示從 1 加到 n 的總和 (1 + 2 + ... + n)。

執行結果 (模擬 Console):

等待執行...

對應程式碼邏輯：

# 輸入必須用 int() 轉換成整數
n = int(input('Please enter the number n:'))
# 計算總和，要先歸零                            
total = 0

# 迴圈：從 1 執行到 n
# i 的型態是整數 (int)，range() 產生從 1 到 n 的數字序列
for i in range(1, n + 1):
    total = total + i   # 縮排，重複執行                     
    # 另一種寫法： total += i 

# print 輸出時，若用 + 號連接，兩邊必須都是字串
# 使用 str() 函式將數字 n 及 total 轉換成字串，                            
print('1+2+...+' + str(n) + '=' + str(total))
# f-string 格式化輸出，較簡潔且易讀，會自動將 {} 內的變數 (不論是 int 或 float) 轉換為字串顯示
#print請選一個輸出方式使用
#print(f'1+2+...+{n}={total}')

💡 觀念提示：

  • for 迴圈：用於重複執行程式碼。range(1, n+1) 代表產生從 1 到 n 的數字序列。
  • 冒號 (:)：`for` 語句結尾必須加上冒號，告訴 Python 接下來是迴圈的內容。
  • 縮排 (Indentation)：迴圈內部的程式碼 (如 total += i) 必須縮排 (1個Tab或4個空白)，表示它們屬於這個迴圈。
  • f-string：字串前加上 f，可以在 {變數} 內直接放入變數，例如 {total}，讓字串組合更直觀。

密碼檢查程式 (while 迴圈)

預設密碼為 137。若第一次輸入錯誤，可再嘗試兩次 (共3次)。三次錯誤則鎖定帳號。

執行結果 (模擬 Console):

等待執行...

對應程式碼邏輯：

password = "137"
times = 1     # 將輸入的次數設為 1 存到變數 times。
password2 = input('Please enter password:\n') 

# 使用 while 迴圈，當密碼不正確且次數小於 3 時執行
while (password != password2) and (times < 3):
    print('Wrong password!')
    times = times + 1   # 錯誤次數加 1
    # 再次輸入密碼
    password2 = input('Please enter password:\n')

# 跳出迴圈表示密碼正確或已達 3 次錯誤

# 使用雙向選擇結構判斷結果
if password2 == password:
    # 當密碼相符，呈現歡迎使用本系統
    print('Welcome to this system.')
else:
    # 當密碼不符，呈現帳號鎖定
    print('Enter the wrong password 3 times, the account is locked.')

💡 觀念提示：

  • while 迴圈：當條件成立時，重複執行區塊內的程式碼。
  • 複合條件 (and)：同時滿足「密碼不對」且「次數還沒到」才會繼續問。
  • 計數器 (Counter)：使用變數 times 紀錄次數，並在每次錯誤後更新它。
