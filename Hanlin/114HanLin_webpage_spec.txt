# 網頁規格說明書：Python 程式設計概念互動網頁 (翰林版)

## 1. 專案概述
- **網頁標題**: Python 程式設計概念互動網頁 (翰林版)
- **對應章節**: 2-2 變數、運算、判斷與迴圈
- **技術架構**: 
  - HTML5
  - Tailwind CSS (使用 CDN 快速排版)
  - Vanilla JavaScript (處理前端互動邏輯)
- **視覺風格**: 
  - 清爽簡約，使用藍、綠、紫、靛等色系區分不同單元。
  - 左側為操作區 (輸入框、按鈕、執行結果)。
  - 右側為程式碼展示區 (暗色背景代碼塊) 與觀念提示。

## 2. 通用功能與佈局
- **導航列 (Tabs)**: 頂部設有 5 個切換按鈕，分別對應不同程式觀念。
- **防止作弊**: 網頁禁用右鍵選單 (`oncontextmenu="return false;"`)，body 設為無法選取文字 (`user-select: none`)，但 input/textarea 除外。
- **互動特色**:
  - **換行符號切換**: 每個單元皆有一個 Checkbox 「在提示字串後加入換行符號 (\n)」，勾選後會改變模擬 Python `input()` 的提示行為及對應程式碼顯示。
  - **模擬 Console**: 執行結果顯示在綠色背景的區塊中，模擬 Python 終端機的文字輸出。
  - **動態程式碼**: 根據是否勾選「換行符號」，右側 Python 程式碼會即時更新 (顯示或隱藏 `\n`)。

## 3. 分頁功能詳解

### Tab 1: 變數與輸入輸出 (Greeting)
- **核心觀念**: `input()`, `print()`, 字串型態 (str)。
- **輸入介面**: 
  - 姓名輸入框 (Text)。
- **邏輯行為**:
  - Python 邏輯: `name = input(...)` -> `print('Hello', name, '!')`。
  - 顯示輸入的姓名與 Hello 字串拼接結果。
- **教學提示**: 強調 `input()` 讀入皆為字串，`print` 用逗號分隔會自動加空格。

### Tab 2: 變數與運算 (Average)
- **核心觀念**: 資料型態轉換 `int()`, `float`, 算術運算。
- **輸入介面**: 
  - 數字 x 輸入框。
  - 數字 y 輸入框。
- **邏輯行為**:
  - Python 邏輯: `int(input(...))` 轉型 -> 計算 `z = (x + y) / 2` -> `print`。
  - 顯示計算後的平均值。
- **教學提示**: 字串無法直接運算，需轉型；平均數除法會產生浮點數。

### Tab 3: 條件判斷 (Grade)
本頁包含兩個區塊：

#### 3-1. 計算學期成績 (if/else)
- **輸入介面**: 作業 (40%), 測驗 (40%), 平時 (20%) 三個成績輸入框。
- **邏輯行為**:
  - 計算加權分數。
  - 判斷 `if grade < 60` 顯示 "Failed!", 否則 (`else`) 顯示 "Pass!"。
- **教學提示**: 縮排的重要性，`if` 與 `else` 的結構。

#### 3-2. 個人平均成績比較 (a051) (if/elif/else)
- **輸入介面**: 國文、英文、數學三個成績輸入框。
- **邏輯行為**:
  - 計算個人平均 `a`。
  - 設定班級平均 `b = 70`。
  - 比較: 
    - `if a > b`: "You are great"
    - `elif a < b`: "Keep on"
    - `else`: "Come on"
- **教學提示**: 多重條件判斷 `elif` 的用法。

### Tab 4: 計次式迴圈 (Loop)
- **核心觀念**: `for` 迴圈, `range()`, 累加計算。
- **輸入介面**: 
  - 數字 n 輸入框 (預設 10)。
- **邏輯行為**:
  - Python 邏輯: `for i in range(1, n + 1): total += i`。
  - 計算 1 加到 n 的總和。
  - 顯示詳細計算步驟 (若 n <= 100)。
- **教學提示**: `range` 的範圍是含頭不含尾，`f-string` 的格式化輸出。

### Tab 5: 條件式迴圈 (Password)
- **核心觀念**: `while` 迴圈, 複合條件 (`and`), 計數器。
- **輸入介面**: 
  - 提供 3 個密碼輸入框，模擬第 1, 2, 3 次嘗試。
- **邏輯行為**:
  - 預設正確密碼 "137"。
  - Python 邏輯: `while (password != input) and (times < 3)`。
  - 模擬逐次檢查：若錯誤且次數未達 3 次則繼續迴圈；若正確或次數滿 3 次則跳出。
  - 最終判斷：密碼正確顯示歡迎，錯誤 3 次顯示鎖定。
- **教學提示**: `while` 執行條件與跳出時機，計數器變數的更新。

## 4. 程式碼維護說明
- 所有的邏輯實作位於 HTML 檔底部的 `<script>` 標籤內。
- 每個 Tab 對應一組 `update...Code()` (更新右側 Python 程式碼顯示) 與 `run...()` (執行 JS 模擬邏輯) 函式。
- 使用 `switchTab()` 函式控制 Tailwind CSS 的 `hidden` 與 `block` class 來切換頁面顯示。
